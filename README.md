# AI Resume Analyzer and Job Matching System

##  Description

An AI-powered web application that analyzes resumes using NLP techniques, calculates ATS scores, and performs job matching using TF-IDF and cosine similarity.

##  Features

* Resume Analysis
* ATS Score Calculation
* Job Matching
* Resume Suggestions
* Resume Builder

##  Tech Stack

* Python
* Django
* NLP (NLTK)
* Scikit-learn

##  How to Run

pip install -r requirements.txt
python manage.py runserver

##  Output

* ATS Score
* Job Match Score
* Resume Suggestions
